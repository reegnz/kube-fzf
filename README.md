# kube-fzf

fzf ❤️  kubectl

# Requirements

* kubectl
* fzf
* pygmentize (for fancy yaml colors)
* bash
* coreutils (sort)
* grep

# Install

Make sure the script is on your path, eg.:

```
mkdir ~/bin
cp kube-fzf ~/bin
```

Then in your bashrc or zshrc put this line:

```
export PATH=$PATH:$HOME/bin
```

# Usage

Just type `kube-fzf`.
